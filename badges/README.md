# badges/

Insignias oficiales de las certificaciones que integra el plan de estudios.
Se muestran en el carrusel de `certifications.html`.

## Convención

- **PNG con fondo transparente**, cuadrado, mínimo 256 × 256 px, menos de 50 KB.
- Deben leerse sobre fondo oscuro (`#081127`): el carrusel va en una sección `dark-bg`.
- El nombre del archivo debe coincidir exactamente con el `src` del HTML.
  Renombrar un archivo aquí rompe la imagen en el sitio.
